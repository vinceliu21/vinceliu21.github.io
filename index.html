Projects

The Projects sections is currently being worked on. Stay tuned or reach out for more information!

4U

4U is a personalized voice assistant that was developed using the IBM Watson SDK for Carnegie Mellon's Tartan Hacks competition. The user first registers their profile with facial recognition. Afterwards, the assistant can be used to help perform various daily activities such as planning their budget or getting food recommendations.

Holographic Arcade Table

Utilizing the Pepper's Ghost illusion technique, our Build18 team built a two player arcade table where users could control holograms and battle against one another. With a budget of $250, we used an acrylic pyramid for the diplsay, a raspberry pi for the graphics, and an arduino Mega for the arcade controls.

Monocular SLAM

I'm very passionate about new techniques in enabling technology to see and perceive the world like humans do. I have extended experience in research and industry working on visual SLAM techniques. The fields I'm particularily focused on are autonomous navigation and VR/AR.

Automated Home AI

I am currently in the process of building a robust system to automate many features -- lights, sound, temperature control -- in my apartment. The system will make use of Jasper's platform and a custom speech to text engine.
