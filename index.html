4U: Personalized Voice Assistant

February 2017

4U is a personalized voice assistant project that was developed during Carnegie Mellon's 2017 Spring Hackathon by myself and three other students. We wanted to levarage the new IBM Watson SDK for Speech to Text (STT) and facial recognition capabilities, therefore we decided that an assistant to help in managing daily activies would be a perfect project.

Key Features

  1. Facial Recognition
    • OpenCV laptop webcam integration to capture user's face
    • IBM Watson Visual Recognition API to train facial recognition model
  2. Restaurant Recommendations
    • Watson STT API to detect "I want to eat ____"
    • Yelp API integration to search local restaurants for the desired food
  3. Budget Planning
    • Watson STT API to allow for user to log budget activity through speech

Zokyo: GitHub Chrome Extension

January 2019

Zokyo was a project to learn the process and intricacies of developing chrome extensions. I decided to build a GitHub chrome extension that helped developers view their GitHub tickets from any chrome tab. Along with viewing current tickets, developers are also be able to open new tickets, close tickets, and also edit tickets.

I launched the project on the Chrome Developer Store and posted on Product Hunt where I was on the front page for that day's product launches. The project is not maintained anymore, but you can find more information here.

CloudTrack: AWS Slack Bot

2019

CloudTrack is a Slack Bot built with Amazon Web Services (AWS) integration to post daily updates about currently live cloud instances and their specifics: type of instance, instance name, and instance zone.

The inspiration for CloudTrack came from the too often occurence of deploying cloud instances during development and then not shutting them down properly afterwards. It is also easy for developers to quickly forget about how many instances they have deployed. There's too many web apps for developers to check, but developers always work in slack and so that's what led to the inspiration for a slack bot interface to post daily updates of cloud instances.

Build18: Holographic Arcade Table

2016

For Carnegie Mellon's 2016 hardware Hackathon, I partnered with three other students to build an arcade table where two users could control holograms and battle one another. The holograms were displayed using an acrylic pyramid, a monitor, and pepper's ghost for the illusion effect. The arcade controls were built using a Raspberry Pi and Arduino Mega, while the game logic was made with Python.

We were awarded the Build 18 Officer's Choice award for the judge's favorite project out of the 80 other presentations.
